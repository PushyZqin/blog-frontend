# Pushy Blog

一个简单的使用Vue搭建的博客前台和管理员后台的单页面项目。

## 前台

前台主要通过Vue + mdui框架搭建，尊崇卡片式与简洁的UI规范，以及良好的手机端适应的效果。以下为部分页面展示图：

## 管理员后台

管理员后台主要也是通过Vue + mdui搭建，有完好的登录机制和路由拦截。主要的功能有：

- 发表文章，具备markdown编辑器器，能够进行在线编辑。

- 管理文章：能够对文章进行更改、删除的操作。

- 管理标签：能够管理文章的标签
